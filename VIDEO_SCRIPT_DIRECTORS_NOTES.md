## General Production Notes

### Color Scheme
- **Primary**: Deep Blue (#1A1F3A) - Trust, security
- **Accent**: Gold (#FFD700) - Premium, attention
- **Success**: Green (#00D084) - Encrypted/secure
- **Warning**: Red (#FF4444) - Problems/mistakes
- **Information**: Cyan (#00BFFF) - Technical info

### Typography
- **Headlines**: Modern, bold, sans-serif
- **Body Text**: Clear, readable, clean font
- **Code**: Monospace, syntax-highlighted

### Audio Design
- **Background Music**: Tech-forward, modern, uplifting
- **Sound Effects**: Subtle, professional, minimal
- **Voiceover**: Clear, professional, enthusiastic pace
- **Transitions**: Smooth, non-intrusive sound design

### Pacing
- **Introduction**: Calm, informative
- **Problem Section**: Slightly faster, emphasizing urgency
- **Solution Section**: Building momentum
- **Technical Demo**: Detailed, allowing time to absorb
- **Features**: Rapid-fire but readable
- **Call to Action**: Clear, confident pace
- **Closing**: Inspiring, memorable

### Camera Work (if using screen recording)
- **Smooth zooms**: On important code sections
- **Pan effects**: Across file structures
- **Highlight animations**: On key concepts
- **Focus shifts**: From overview to details

### Text Display Timing
- Each text element: 3-4 seconds on screen
- Code snippets: Full reading time + buffer
- Statistics: Clear visibility (2-3 seconds)
- Calls to action: Prominent, readable

### Logo Placement
- **Zama Logo**: Top-right, semi-transparent during content
- **Watermark**: Bottom-right for entire duration
- **Final Slide**: Centered, prominent

### Subtitle/Caption Options
- English subtitles throughout
- Optional: Key terms defined in captions
- Code examples: Line-by-line captions available

## Technical Specifications

### Video Format
- **Resolution**: 1920x1080 (Full HD) minimum
- **Frame Rate**: 30fps (24fps for cinematic look)
- **Codec**: H.264 or ProRes for editing
- **Aspect Ratio**: 16:9 (standard)

### Audio Specifications
- **Sample Rate**: 48kHz
- **Bit Depth**: 16-bit
- **Format**: Stereo
- **Voiceover**: Professional quality
- **Background Music**: Royalty-free or licensed

### Animation Specifications
- **Transition Duration**: 0.5-1.0 seconds
- **Text Fade**: 0.3 seconds
- **Scene Transitions**: Smooth cross-dissolve
- **Emphasis Effects**: Subtle scale/glow

### Expected Timing
- Total Duration: 70-75 seconds (as required)
- Scene Breakdown: See above
- Buffer for transitions: Built into timings
